# Reference Assets

This directory stores static reference material used to keep the product implementation aligned with the intended design language.

## Current Asset

- `nexus-class-prototype.html`
  - The original Apple-style visual prototype used as the frontend baseline.

## Usage

Use these files as visual references only. They are not the production source of truth for routing, state, security, or backend integration.

If a future agent updates the visual direction, keep the old artifact unless it is explicitly obsolete and replaced by a newer approved reference.
